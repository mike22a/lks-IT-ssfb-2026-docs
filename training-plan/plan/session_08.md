# Sesi 8 - Review & Practice

## Informasi Sesi
- **Durasi**: 2 jam (120 menit)
- **Fokus**: Review semua materi dan practice
- **Prerequisite**: Selesai Sesi 7, API integration di mobile sudah berfungsi, semua materi sebelumnya sudah dipelajari

## Tujuan Pembelajaran
- [ ] Siswa dapat mereview semua materi yang sudah dipelajari
- [ ] Siswa dapat mengintegrasikan semua modul (API, Desktop, Mobile)
- [ ] Siswa dapat melakukan full integration practice
- [ ] Siswa dapat mempraktikkan time management
- [ ] Siswa siap untuk kompetisi

## Materi (120 menit)

### Review Homework (15 menit)
- Review CRUD implementation di mobile
- Review form validation
- Review navigation
- Feedback dan koreksi
- Q&A dari homework

### Review API .NET (20 menit)
- Review RESTful API concepts
- Review CRUD endpoints
- Review business logic implementation
- Review error handling
- Q&A untuk API

### Review Desktop Apps (20 menit)
- Review Windows Forms concepts
- Review Entity Framework integration
- Review CRUD operations
- Review advanced queries dan charts
- Q&A untuk Desktop

### Review Mobile Android (20 menit)
- Review Jetpack Compose concepts
- Review API integration di mobile
- Review state management
- Review CRUD di mobile
- Q&A untuk Mobile

### Full Integration Practice (30 menit)
- Latihan integrasi semua modul
- Time-boxed: 30 menit
- Simulasi pressure kompetisi
- Test end-to-end flow

### Competition Simulation (10 menit)
- Mini competition simulation
- Time management practice
- Prioritization practice
- Q&A dan clarification

### Final Review (5 menit)
- Final assessment
- Identify areas untuk improvement
- Competition strategy tips
- Encouragement

## Homework

### Tugas 1: Final Practice
- Lakukan full practice run untuk semua modules (API, Desktop, Mobile)
- Time yourself: 3 jam untuk desktop, 3 jam untuk mobile
- Identify weak areas
- Practice weak areas
- **Deadline**: Sebelum kompetisi

### Tugas 2: Review Competition Requirements
- Baca ulang semua competition requirements
- Pastikan semua criteria terpenuhi
- Checklist semua requirements
- **Deadline**: Sebelum kompetisi

### Tugas 3: Mental Preparation
- Review time management strategy
- Plan approach untuk competition day
- Prepare checklist untuk competition day
- **Deadline**: Sebelum kompetisi

## Notes untuk Mentor

### Tips
- Fokus pada review semua materi secara komprehensif
- Simulasikan pressure kompetisi
- Emphasize time management
- Provide final feedback dan encouragement
- Jawab semua pertanyaan siswa

### Common Pitfalls
- Siswa lupa materi dari sesi awal
- Integration antar modul tidak berfungsi
- Time management tidak proper
- Validasi dan error handling terlupakan

### Troubleshooting
- Review materi yang sering dilupakan
- Test integration end-to-end
- Practice time management dengan timer
- Have backup plan untuk technical issues

### Final Preparation
- Review semua homework dari semua sesi
- Identify common mistakes
- Provide tips untuk competition day
- Encourage siswa untuk practice lebih lanjut
- Pastikan siswa siap secara mental dan teknis

## Competition Day Checklist

### Sebelum Kompetisi
- [ ] Semua software terinstal dan berfungsi
- [ ] API berjalan dan dapat diakses
- [ ] Database sudah siap dengan sample data
- [ ] Desktop apps sudah lengkap dan tested
- [ ] Mobile apps sudah lengkap dan tested
- [ ] Semua requirements terpenuhi
- [ ] Time management strategy sudah dipersiapkan

### Selama Kompetisi
- [ ] Baca requirements dengan teliti
- [ ] Prioritaskan features berdasarkan weight
- [ ] Manage time secara efektif
- [ ] Test semua functionality sebelum submit
- [ ] Validasi dan error handling tidak diabaikan
- [ ] Format tanggal sesuai ISO (YYYY-MM-DD)
- [ ] Formulir terpusat di layar
- [ ] Scrollbar ditampilkan jika perlu

### Setelah Kompetisi
- [ ] Review hasil
- [ ] Identify areas for improvement
- [ ] Document lessons learned
