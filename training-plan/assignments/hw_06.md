## Tugas 1: Complete Customer Form di Android

### Deskripsi
Buat customer form yang lengkap dengan state management dan validation.

### Requirements
1. **Form Design**
   - Buat composable function untuk customer form
   - Gunakan TextField untuk:
     - CustomerName (wajib)
     - Email (wajib)
     - Phone (opsional)
     - Address (opsional)
   - Gunakan Button untuk:
     - Save
     - Cancel

2. **Implementasi State Management**
   - Gunakan `remember` dan `mutableStateOf` untuk setiap field
   - Implementasi state hoisting jika perlu
   - Update state saat user input
   - Test state updates

3. **Implementasi Validation**
   - Validasi field tidak boleh kosong (Name, Email)
   - Validasi format email
   - Display error messages di UI
   - Disable Save button jika invalid

4. **Implementasi Submit Button**
   - Validasi input saat submit
   - (Untuk sekarang) Log input ke console atau display di UI
   - Clear form setelah save
   - Navigate back atau show success message

### Deliverables
- [ ] Customer form composable
- [ ] State management implementation
- [ ] Validation implementation
- [ ] Screenshot form di emulator

## Tugas 3: Create Customer List Screen

### Deskripsi
Buat list screen untuk menampilkan customers dengan LazyColumn.

### Requirements
1. **List Screen Design**
   - Buat composable function untuk customer list
   - Gunakan LazyColumn untuk display list
   - Gunakan item composable untuk setiap customer
   - Add FAB (Floating Action Button) untuk add customer

2. **Mock Data**
   - Create mock data untuk customers
   - Load mock data ke LazyColumn
   - Display customer info di item
   - (Untuk sekarang) Gunakan hardcoded data

3. **Item Composable**
   - Buat item composable untuk customer
   - Display customer name, email, phone
   - Add click handler untuk item
   - Style dengan Card atau Surface

4. **Navigation**
   - Navigate ke customer form saat FAB diklik
   - Navigate ke customer form saat item diklik (untuk edit)
   - Implementasi back navigation

### Deliverables
- [ ] Customer list composable
- [ ] LazyColumn implementation
- [ ] Item composable
- [ ] Navigation implementation
- [ ] Screenshot list di emulator
