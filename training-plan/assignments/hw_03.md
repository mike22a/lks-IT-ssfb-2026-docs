## Tugas 1: Create Customer Form

### Deskripsi
Buat form untuk input customer data dengan proper validation.

### Requirements
1. **Form Design**
   - Buat Windows Form baru
   - Tambahkan TextBox untuk:
     - Customer Name (wajib)
     - Email (wajib, valid format)
     - Phone (opsional)
     - Address (opsional)
   - Tambahkan Button untuk:
     - Save
     - Cancel
   - Center form di layar

2. **Implementasi Validation**
   - Validasi field tidak boleh kosong (Name, Email)
   - Validasi format email
   - Display error messages yang jelas
   - Disable Save button jika invalid
   - Atau tampilkan error saat Save diklik

3. **Implementasi Button Handlers**
   - Save button:
     - Validasi input
     - Jika valid, kirim data ke API (POST)
     - Tampilkan success/error message
     - Clear form setelah save
   - Cancel button:
     - Clear form
     - Atau close form

### Deliverables
- [ ] Customer Form dengan semua controls
- [ ] Validation implementation
- [ ] Button handlers implementation
- [ ] Screenshot form

## Tugas 2: Create Customer List Form

### Deskripsi
Buat form untuk menampilkan list customer dengan CRUD operations.

### Requirements
1. **Form Design**
   - Buat Windows Form baru
   - Tambahkan DataGridView untuk display customer list
   - Tambahkan Button untuk:
     - Add (buka form input)
     - Edit (buka form edit dengan data terpilih)
     - Delete (hapus customer terpilih)
     - Refresh (reload data dari API)
   - Center form di layar

2. **Implementasi Data Loading**
   - Load customer data dari API (GET)
   - Bind data ke DataGridView
   - Auto-generate columns atau manual columns
   - Format columns (date, currency, dll. jika ada)

3. **Implementasi Button Handlers**
   - Add button:
     - Buka Customer Form (mode add)
     - Refresh list setelah save
   - Edit button:
     - Buka Customer Form (mode edit)
     - Load data terpilih ke form
     - Refresh list setelah update
   - Delete button:
     - Confirm delete (MessageBox)
     - Call API DELETE
     - Refresh list setelah delete
   - Refresh button:
     - Reload data dari API
     - Update DataGridView

4. **Implementasi Navigation**
   - Buka Customer List Form dari Main Form
   - Buka Customer Form dari Customer List Form
   - Pass data antar forms jika perlu

### Deliverables
- [ ] Customer List Form dengan DataGridView
- [ ] Data loading implementation
- [ ] Button handlers implementation
- [ ] Navigation implementation
- [ ] Screenshot form

## Tips

### Customer Form
- Gunakan ErrorProvider untuk validation feedback
- Center form dengan StartPosition.CenterParent
- Gunakan proper naming conventions untuk controls
- Test validation dengan berbagai scenarios

### Customer List Form

- Handle loading states (show loading indicator)
- Handle errors dengan user-friendly messages
- Refresh DataGridView setelah operations
