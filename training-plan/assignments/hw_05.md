## Tugas 1: Practice Run — Buat Ulang dari Awal (Wajib)

Ini adalah tugas paling penting sebelum kompetisi.

### Instruksi:
Buat ulang sistem lengkap (API + WinForm + Android) dari **awal** tanpa melihat kode sebelumnya, menggunakan domain bisnis baru yang belum pernah dipakai.

**Contoh domain baru yang bisa dipilih:**
- Library Management: `Book`, `Member`, `Loan`
- Hospital: `Doctor`, `Patient`, `Appointment`
- Restaurant: `Menu`, `Table`, `Order`, `OrderItem`

### Target Waktu:
| Komponen | Target Waktu |
|----------|-------------|
| API (Clean Architecture) skeleton + CRUD + Business Logic | 90 menit |
| WinForm CRUD via API | 60 menit |
| Android List + minimal POST | 45 menit |
| Total | ~3 jam 15 menit |

### Rubrik Self-Assessment:
- [ ] **API**: Solution dengan 4 project terbuat dalam 20 menit
- [ ] **API**: Database + migration berhasil dalam 30 menit
- [ ] **API**: Semua CRUD endpoint berjalan di Swagger dalam 90 menit
- [ ] **WinForm**: DataGridView load dari API dalam 20 menit setelah mulai WinForm
- [ ] **WinForm**: CRUD lengkap via API dalam 60 menit
- [ ] **Android**: List tampil dari API dalam 30 menit setelah mulai Android
- [ ] **Semua**: End-to-end test berhasil

> Jika ada bagian yang memakan waktu jauh lebih lama dari target, itu sinyal area yang perlu lebih banyak latihan.

## Tugas 3: Persiapan Teknis (Wajib)

Lakukan verifikasi teknis lengkap di komputer yang akan digunakan saat kompetisi.

### Checklist Software:
- [ ] Visual Studio bisa membuka solution dan build tanpa error
- [ ] `dotnet new webapi` berjalan di terminal
- [ ] `dotnet ef migrations add Test` berjalan (lalu hapus migration-nya)
- [ ] SQL Server service berjalan
- [ ] SSMS bisa connect ke SQL Server
- [ ] Postman terinstall dan bisa request ke localhost
- [ ] Android Studio bisa membuka project Android
- [ ] Emulator Android bisa berjalan (AVD Manager → Start)
- [ ] Emulator bisa akses API di `http://10.0.2.2:5000`

### Buat Contekan (Cheat Sheet):
Buat 1 halaman ringkasan yang berisi:
1. Struktur folder Clean Architecture (project apa saja, referensi ke mana)
2. Commands `dotnet ef` yang sering dipakai
3. Template `ApiResponse<T>` class
4. Template `ExceptionHandlingMiddleware` minimal
5. Template Retrofit setup Android (RetrofitClient + ApiService)
6. Template HttpClient setup WinForm (ApiClient)

> Di kompetisi kamu boleh bawa catatan fisik atau dokumen digital (tergantung aturan). Siapkan dari sekarang.

## Tugas 4: Mental Preparation (Opsional tapi Penting)

### Review Strategi Kompetisi:
1. **Urutan pengerjaan**: API → WinForm → Android
2. **Time boxing per modul**: API 90', WinForm 90', Android 60', Polish 30'
3. **MVP mindset**: selesaikan yang pasti dinilai dulu, bukan yang sempurna
4. **Test berkala**: setiap 20-30 menit test yang sudah dibuat

### Mindset:
- Kompetisi adalah tentang menunjukkan apa yang **sudah kamu bisa**, bukan semua yang ada di soal
- Error dan bug itu normal — yang membedakan adalah kemampuan debugging cepat
- 10 menit baca soal dengan teliti lebih berharga dari 1 jam coding yang salah arah
