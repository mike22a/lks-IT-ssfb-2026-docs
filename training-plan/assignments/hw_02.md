## Tugas 1: Lengkapi Semua Service (Wajib)

Semua entity harus memiliki Service layer yang lengkap.

### Persyaratan per Entity:
1. **DTO lengkap** di `Application/DTOs/`:
   - `XxxDto` — response object
   - `CreateXxxRequest` — untuk POST
   - `UpdateXxxRequest` — untuk PUT

2. **Service Interface** di `Application/Interfaces/`:
   ```csharp
   public interface IProductService
   {
       Task<IEnumerable<ProductDto>> GetAllAsync();
       Task<ProductDto?> GetByIdAsync(int id);
       Task<ProductDto> CreateAsync(CreateProductRequest request);
       Task<ProductDto> UpdateAsync(int id, UpdateProductRequest request);
       Task DeleteAsync(int id);
   }
   ```

3. **Service Implementation** di `Application/Services/`:
   - Constructor inject `IRepository<T>` (atau specific interface)
   - Implementasi mapping manual: Entity → Dto
   - Throw `NotFoundException` jika `GetByIdAsync` return null

4. **Custom Exception** di `Domain/Exceptions/`:
   ```csharp
   public class NotFoundException : Exception
   {
       public NotFoundException(string message) : base(message) { }
   }
   ```

5. **Dependency Injection** — daftarkan semua Service di `Program.cs`

### Deliverable:
- Semua entity memiliki DTO + Service Interface + Service Implementation
- `NotFoundException` ada di Domain layer
- Semua terdaftar di DI (Program.cs)
- `dotnet build` sukses tanpa error

## Tugas 2: Seed Data (Wajib)

Tambahkan data awal ke database agar testing lebih mudah.

### Cara: Gunakan `HasData` di `OnModelCreating`
```csharp
protected override void OnModelCreating(ModelBuilder modelBuilder)
{
    modelBuilder.Entity<Product>().HasData(
        new Product { Id = 1, Name = "Laptop", CategoryId = 1, Price = 15000000, Stock = 10, CreatedAt = DateTime.UtcNow },
        new Product { Id = 2, Name = "Mouse", CategoryId = 1, Price = 150000, Stock = 50, CreatedAt = DateTime.UtcNow },
        new Product { Id = 3, Name = "Keyboard", CategoryId = 1, Price = 300000, Stock = 30, CreatedAt = DateTime.UtcNow },
        new Product { Id = 4, Name = "Monitor", CategoryId = 1, Price = 3500000, Stock = 5, CreatedAt = DateTime.UtcNow },
        new Product { Id = 5, Name = "Headset", CategoryId = 1, Price = 250000, Stock = 20, CreatedAt = DateTime.UtcNow }
    );
    // ... tambahkan seed untuk entity lainnya
}
```

### Langkah:
```bash
# Buat migration baru untuk seed data
dotnet ef migrations add SeedData --project BusinessApp.Infrastructure --startup-project BusinessApp.API

# Apply migration
dotnet ef database update --project BusinessApp.Infrastructure --startup-project BusinessApp.API
```

### Verifikasi:
- Buka SSMS → cek tabel — data harus sudah ada
- Jalankan API → GET endpoint — data harus tampil

### Deliverable:
- Minimal **5 rekaman** seed data per entity
- Data sudah ada di database (screenshot SSMS)

## Tugas 3: Pelajari HTTP Status Codes (Wajib)

Hafal dan pahami kapan menggunakan status code berikut:

| Status Code | Nama | Kapan Digunakan |
|-------------|------|-----------------|
| 200 | OK | GET berhasil, PUT berhasil |
| 201 | Created | POST berhasil membuat resource baru |
| 204 | No Content | DELETE berhasil |
| 400 | Bad Request | Input tidak valid (format salah) |
| 404 | Not Found | Resource tidak ditemukan |
| 409 | Conflict | Duplikat data (misal: email sudah ada) |
| 422 | Unprocessable Entity | Business rule dilanggar (misal: stok tidak cukup) |
| 500 | Internal Server Error | Error tak terduga di server |

Catat pertanyaan dan bawa ke sesi 3.
