# Homework 04 - Desktop Apps Database Integration

## Informasi
- **Sesi**: 4
- **Deadline**: Sesi 5
- **Estimasi Waktu**: 2-3 jam
- **Fokus**: CRUD Operations, DataGridView Customization, Error Handling

## Tugas 1: Complete CRUD untuk Semua Entities

### Deskripsi
Implementasi CRUD operations untuk semua entities dari database.

### Requirements
1. **Create Form Input untuk Setiap Entity**
   - Buat form input untuk setiap entity dari database
   - Gunakan TextBox, ComboBox, DateTimePicker sesuai tipe data
   - Implementasi validation di form
   - Center form di layar

2. **Implementasi CRUD Operations**
   - **Create**: Form input untuk insert data baru
   - **Read**: DataGridView untuk display data
   - **Update**: Form edit untuk update data terpilih
   - **Delete**: Konfirmasi delete dan hapus data

3. **Integrasikan dengan API**
   - Gunakan API yang sudah dibuat di sesi 1-2
   - POST untuk create
   - GET untuk read
   - PUT untuk update
   - DELETE untuk delete

4. **Test Semua CRUD Operations**
   - Test create dengan valid data
   - Test read (load data ke DataGridView)
   - Test update dengan berbagai scenarios
   - Test delete dengan konfirmasi

### Deliverables
- [ ] Form input untuk semua entities
- [ ] CRUD implementation untuk semua entities
- [ ] API integration successful
- [ ] Test results

## Tugas 2: DataGridView Customization

### Deskripsi
Customize DataGridView untuk better user experience.

### Requirements
1. **Custom Column Types**
   - Gunakan ComboBox column untuk foreign keys
   - Gunakan Button column untuk actions (Edit, Delete)
   - Gunakan CheckBox column untuk boolean fields
   - Gunakan Image column jika perlu

2. **Column Formatting**
   - Format date columns (YYYY-MM-DD)
   - Format currency columns (dengan simbol Rp)
   - Format numeric columns (decimal places)
   - Set column widths yang tepat

3. **Sorting dan Filtering**
   - Enable sorting untuk columns
   - Implementasi filter functionality
   - Filter dialog atau filter bar
   - Multiple filter criteria

4. **Context Menu**
   - Add context menu untuk DataGridView
   - Menu items: Edit, Delete, Refresh, Export
   - Implementasi handlers untuk setiap menu item

### Deliverables
- [ ] DataGridView dengan custom columns
- [ ] Column formatting
- [ ] Sorting dan filtering implementation
- [ ] Context menu implementation
- [ ] Screenshot DataGridView

---

## Tugas 3: Error Handling di Desktop

### Deskripsi
Implementasi proper error handling untuk semua API calls di desktop.

### Requirements
1. **Implementasi Try-Catch untuk Semua API Calls**
   - Wrap semua API calls dengan try-catch
   - Catch specific exceptions (HttpRequestException, TaskCanceledException, dll.)
   - Log errors untuk debugging (jika diizinkan)

2. **Display User-Friendly Error Messages**
   - Gunakan MessageBox untuk error display
   - Error messages dalam Bahasa Indonesia
   - Error messages yang jelas dan actionable
   - Different icons untuk error types (Error, Warning, Info)

3. **Handle Network Errors**
   - Detect network connectivity issues
   - Display appropriate message jika offline
   - Implementasi retry mechanism (opsional)
   - Handle timeout errors

4. **Handle Validation Errors dari API**
   - Parse error response dari API
   - Display validation errors di UI
   - Highlight fields dengan errors
   - Show error messages yang spesifik

### Deliverables
- [ ] Error handling implementation di semua API calls
- [ ] User-friendly error messages
- [ ] Network error handling
- [ ] Validation error handling
- [ ] Test results untuk error scenarios

## Tips

### CRUD Operations
- Gunakan async/await untuk API calls
- Refresh DataGridView setelah operasi
- Implementasi confirmation untuk delete
- Validate input sebelum kirim ke API

### DataGridView Customization
- Gunakan AutoGenerateColumns = false untuk manual control
- Set HeaderText untuk column headers
- Use DataGridViewCellStyle untuk formatting
- Handle CellClick untuk button columns

### Error Handling
- Use specific exception types
- Don't expose sensitive information
- Log technical errors untuk debugging
- Provide actionable error messages
