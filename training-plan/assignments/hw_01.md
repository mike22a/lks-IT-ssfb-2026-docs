## Tugas 1: Complete Database Design

### Deskripsi
Design database schema untuk 3-4 entities berdasarkan requirement kompetisi.

### Requirements
1. **Identifikasi Entities**
   - Identifikasi 3-4 entities dari requirement kompetisi
   - Contoh: Customer, Product, Order, OrderDetail

2. **Buat ERD (Entity Relationship Diagram)**
   - Gambar hubungan antar entities
   - Tentukan primary keys dan foreign keys
   - Tentukan cardinality (1:1, 1:N, N:M)
   - Bisa menggunakan Draw.io atau tools lain

3. **Create Tables di SQL Server**
   - Gunakan SQL Server Management Studio (SSMS)
   - Create semua tables berdasarkan ERD
   - Define constraints (NOT NULL, UNIQUE, FOREIGN KEY)
   - Set data types yang tepat

4. **Insert Sample Data**
   - Insert 5-10 rekaman per table
   - Pastikan data valid dan konsisten
   - Test relationships dengan queries

### Deliverables
- [ ] ERD diagram (gambar atau file)
- [ ] SQL scripts untuk create tables
- [ ] SQL scripts untuk insert sample data
- [ ] Screenshot tables dengan data

## Tugas 2: Create Multiple Endpoints

### Deskripsi
Buat controller untuk semua entities dan implementasi GET endpoints.

### Requirements
1. **Create API Project**
   - Gunakan project yang sudah dibuat di sesi 1
   - Atau buat project baru jika perlu

2. **Create Entity Classes**
   - Buat C# class untuk setiap entity
   - Map properties ke columns di database
   - Gunakan data types yang tepat

3. **Create DbContext**
   - Setup DbContext dengan semua entities
   - Configure connection string
   - Test database connection

4. **Create Controllers**
   - Buat controller untuk setiap entity
   - Implementasi GET endpoint untuk setiap entity
   - Return data dalam format JSON

5. **Test Endpoints**
   - Test semua endpoints dengan Postman
   - Verify response format
   - Verify data yang dikembalikan

6. **Document Response Format**
   - Document JSON structure untuk setiap endpoint
   - Include field names dan data types
   - Include example response

### Deliverables
- [ ] Entity classes (C# code)
- [ ] DbContext configuration
- [ ] Controller classes dengan GET endpoints
- [ ] Postman collection dengan requests
- [ ] Response format documentation

## Tips

### Database Design
- Mulai dengan entities yang sederhana
- Gunakan naming conventions yang konsisten
- Pastikan foreign keys terdefinisi dengan benar
- Test relationships dengan JOIN queries

### API Endpoints
- Gunakan RESTful naming conventions
- Return appropriate status codes
- Include error handling basic
- Test dengan Postman secara sistematis

### Study Resources
- Fokus pada concepts yang relevan dengan kompetisi
- Buat catatan untuk reference
- Siapkan pertanyaan untuk sesi berikutnya
