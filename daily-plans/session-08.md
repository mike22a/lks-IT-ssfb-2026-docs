# Sesi 8 - Mobile Android CRUD & Practice

## Informasi Sesi
- **Durasi**: 2 jam (120 menit)
- **Fokus**: Mobile Android
- **Prerequisite**: Selesai Sesi 7, API integration di mobile sudah berfungsi

## Tujuan Pembelajaran
- [ ] Siswa dapat mengimplementasikan CRUD lengkap di mobile
- [ ] Siswa dapat mengimplementasikan search dan filter di mobile
- [ ] Siswa dapat mengimplementasikan data export (CSV/PDF)
- [ ] Siswa dapat melakukan full integration practice
- [ ] Siswa siap untuk kompetisi

## Materi (120 menit)

### Review Homework (15 menit)
- Review CRUD implementation di mobile
- Review form validation
- Review navigation
- Feedback dan koreksi
- Q&A dari homework

### Teori (15 menit)
#### Search dan Filter di Mobile (10 menit)
- Search bar implementation
- Debouncing search input
- Filter dialog
- Multiple filter criteria

#### Data Export (5 menit)
- CSV export di Android
- PDF export di Android
- File permissions
- Export UI

### Demo (30 menit)
#### Search Implementation (10 menit)
- Create search bar composable
- Implementasi debouncing
- Filter list berdasarkan search query
- Update UI dengan filtered results

#### Filter Implementation (10 menit)
- Create filter dialog
- Implementasi multiple filter criteria
- Apply filter ke list
- Reset filter

#### Data Export (10 menit)
- Implementasi CSV export
- Implementasi PDF export (basic)
- Handle file permissions
- Create export UI

### Practice (50 menit)
#### Latihan 1: Full Integration Practice (50 menit)
- Complete CRUD untuk semua entities di mobile
- Implementasi search dan filter
- Implementasi data export
- Test semua functionality
- Time-boxed: 50 menit (simulasi kompetisi)

### Review (10 menit)
- Review hasil latihan siswa
- Final assessment
- Identify areas for improvement
- Competition strategy tips
- Q&A

## Homework

### Tugas 1: Final Practice
- Lakukan full practice run untuk semua modules (API, Desktop, Mobile)
- Time yourself: 3 jam untuk desktop, 3 jam untuk mobile
- Identify weak areas
- Practice weak areas
- **Deadline**: Sebelum kompetisi

### Tugas 2: Review Competition Requirements
- Baca ulang semua competition requirements
- Pastikan semua criteria terpenuhi
- Checklist semua requirements
- **Deadline**: Sebelum kompetisi

### Tugas 3: Mental Preparation
- Review time management strategy
- Plan approach untuk competition day
- Prepare checklist untuk competition day
- **Deadline**: Sebelum kompetisi

## Notes untuk Mentor

### Tips
- Fokus pada integration semua features
- Simulasikan pressure kompetisi
- Emphasize time management
- Provide final feedback dan encouragement

### Common Pitfalls
- Tidak selesai dalam waktu
- Validasi tidak comprehensive
- Error handling tidak proper
- UI tidak user-friendly

### Troubleshooting
- Practice time management
- Prioritize features based on weight
- Test all scenarios
- Have backup plan

### Final Preparation
- Review semua homework dari semua sesi
- Identify common mistakes
- Provide tips untuk competition day
- Encourage siswa untuk practice lebih lanjut
